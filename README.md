🐾 IoT-Based Pet Tracking System (Backend)

📌 Project Overview

This project is a backend system developed for an IoT-based pet tracking solution. It processes GPS data from the device, manages communication, and stores data securely.

🛠 Technologies Used

- Node.js
- Express.js
- Firebase
- MQTT Protocol
- JavaScript

⚙️ Features

- REST API for pet tracking
- Real-time data handling
- Firebase database integration
- Secure communication

📡 System Functionality

The backend receives data from the IoT device, processes it, and stores it in the database. It then sends the data to the frontend application for user interaction.

🚀 How to Run

1. Install dependencies: npm install
2. Run server: node server.js
